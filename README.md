# 一、基于vue3.0+ts+后台admin模板


# 二、下载使用：
> ## 1、克隆代码
>> ```
>> 通过git将代码克隆到本地；或者使用下载安装包模式进行下载。
>> ```

> ## 2、进入目录
>> ```
>> 进入项目的根目录：vue3.0-ts-admin
>> ```

> ## 3、安装依赖
>> ```
>> npm install
>> ```

> ## 4、启动项目
>> ```
>> npm run dev
>> ```
